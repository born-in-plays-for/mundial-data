# mundial-data

Conceptually a subfolder of [mundial-build](https://github.com/born-in-plays-for/mundial-build), promoted to a standalone repo solely to sidestep the complexities of Git's sparse submodule support when sharing data with [mundial](https://github.com/born-in-plays-for/mundial).

Documentation on file contents and usage lives in the respective project READMEs:

- [mundial-build](https://github.com/born-in-plays-for/mundial-build#readme) — data pipeline (generates these files)
- [mundial](https://github.com/born-in-plays-for/mundial#readme) — frontend (consumes these files)
